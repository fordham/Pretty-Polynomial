 polynomial.rb
 Polynomial

Synopsis

Converts a polynomial provided as an array of it‘s coefficients to a pretty-print of the polynomial - [An, An-1, …] => An(x^n) + An-1(x^n-1)+…

Author

 Fred Fordham on 01/12/2009.

Usage

 puts Polynomial.new([-3,-4,1,0,6]) # => -3x^4-4x^3+x^2+6
 puts Polynomial.new([1,0,2]) # => x^2+2
